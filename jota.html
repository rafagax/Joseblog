¿Qué es una etiqueta HTML?

15 de Julio 2024

Una etiqueta HTML es un fragmento de código que define la estructura y el contenido de una página web. Las etiquetas son fundamentales para crear y diseñar páginas web, ya que indican a los navegadores cómo deben mostrar el contenido.

Ejemplos de etiquetas HTML

8 de Julio 2024

Algunos ejemplos comunes de etiquetas HTML son <h1>, <p>, <a>, entre otras. La etiqueta <h1> se utiliza para definir el encabezado principal, <p> para definir un párrafo y <a> para crear enlaces.
